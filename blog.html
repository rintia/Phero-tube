Blog Questions and Answers

1. Discuss the scope of var, let and const.

Ans: "var" declarations are globally scoped or function scoped while "let" and "const" are block scoped.
"var" variables can be updated and re-declared within in it's scope."let" variables can be updated but not re-declared. "const" variables can neither be updated nor re-declared.

2. The use of 'null' and 'undefined'.

Ans: 'null' is special value that represents an empty or special value. 'null' is used for referring absence of any object value.
Where 'undefined' indicates a variable has been declared but has not yet been assigned a value.

3. What do you mean by REST API?

Ans: REST(Representational State Transfer) API is a way of accessing web services in a simple and flexible way. In HTTP there are five methods that are commonly used in a REST-based architecture. The are POST, GET, PUT, PATCH and DELETE. These correspond to create, read, update, and delete (or CRUD) operations respectively. A request is sent from client to server in the form of a web URL as HTTP GET or POST or PUT or PATCH or DELETE request. After that, a response comes back from the server in the form of JSON.
